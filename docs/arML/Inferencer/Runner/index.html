val run_expr_inferencer : Ast.AbstractSyntaxTree.expression -> (TypeTree.typ, TypeErrors.error) Stdlib.result

Runs the type inference process on a given expression and returns either the inferred type or a type error.

val run_program_inferencer : Ast.AbstractSyntaxTree.declaration list -> ((string, Schema.schema, Base.String.comparator_witness) Base.Map.t * string list, TypeErrors.error) Stdlib.result

Runs the type inference process on a program represented as a list of structure items. Returns either a mapping of inferred types for named entities along with a list of declarations names, or a type error.
