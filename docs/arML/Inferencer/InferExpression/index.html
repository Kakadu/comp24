val infer_expr : TypeEnv.t -> Ast.AbstractSyntaxTree.expression -> (Substitution.t * TypeTree.typ, TypeErrors.error) Common.StateResultMonad.t

Infers the type of a given expression within the specified type environment. Returns a substitution (mapping of type variables) and the inferred type, or a type error if inference fails.
